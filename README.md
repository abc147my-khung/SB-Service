# S&B SERVICE — เว็บไซต์พร้อมระบบหลังบ้าน

## เปิดใช้งานบนเครื่อง

1. ติดตั้ง Node.js รุ่น 18 ขึ้นไป หากเครื่องยังไม่มี
2. ดับเบิลคลิกไฟล์ `start.bat`
3. เปิดเว็บไซต์ที่ http://127.0.0.1:3000
4. เปิดระบบหลังบ้านที่ http://127.0.0.1:3000/admin

เมื่อเปิดครั้งแรก ระบบจะสร้างชื่อผู้ใช้และรหัสผ่านให้อัตโนมัติ ดูได้ที่ไฟล์:

`data/initial-admin.txt`

หลังเข้าสู่ระบบ ควรเปลี่ยนรหัสผ่านในเมนู “ความปลอดภัย” ไฟล์รหัสผ่านเริ่มต้นจะถูกลบให้อัตโนมัติ

## ความสามารถที่เพิ่มแล้ว

- หน้าภาพรวมจำนวนลูกค้าและสินค้า
- รับข้อมูลลูกค้าจากแบบฟอร์มหน้าเว็บไซต์
- ค้นหา เปลี่ยนสถานะ และลบข้อมูลลูกค้า
- เพิ่ม แก้ไข ซ่อน และลบสินค้า
- ตั้งค่าชื่อธุรกิจ เบอร์โทร LINE และข้อความประกาศ
- เปลี่ยนรหัสผ่านผู้ดูแล
- ป้องกันการเดารหัสผ่านและการส่งฟอร์มถี่เกินไป
- เก็บรหัสผ่านแบบเข้ารหัส และใช้คุกกี้เข้าสู่ระบบแบบ HttpOnly

## ข้อมูลระบบ

ข้อมูลทั้งหมดอยู่ใน `data/store.json` ควรสำรองไฟล์นี้เป็นประจำ และห้ามนำไฟล์ในโฟลเดอร์ `data` ไปเผยแพร่บนเว็บสาธารณะโดยตรง

หากต้องการเปลี่ยนพอร์ต ให้เปิด Command Prompt ในโฟลเดอร์นี้แล้วใช้:

`set PORT=8080 && node server.js`

## ก่อนนำขึ้นใช้งานจริง

- ใช้ HTTPS ผ่านผู้ให้บริการโฮสติ้งหรือ reverse proxy
- ตั้งค่าการสำรอง `data/store.json`
- เปลี่ยนรหัสผ่านเริ่มต้นทันที
- สำหรับผู้ใช้พร้อมกันจำนวนมาก ควรเปลี่ยนจากไฟล์ JSON เป็นฐานข้อมูล PostgreSQL หรือ MySQL
